# AstrBot 番茄钟插件

一个可通过 QQ 群机器人开启、关闭和运行的 AstrBot 番茄钟插件。

## 功能

- 群内开启/关闭番茄钟功能
- 群内开始/停止番茄钟循环
- 命令回复会自动 @ 触发命令的人
- 多个群独立运行
- 插件卸载时自动清理后台任务
- 状态持久化到 `data/state.json`
- 支持 WebUI 配置番茄钟时长

## 命令

### 英文命令

```text
/pomodoro on       开启当前群番茄钟功能
/pomodoro off      关闭当前群番茄钟功能，并停止正在运行的任务
/pomodoro start    开始当前群番茄钟循环
/pomodoro stop     停止当前群番茄钟循环，功能仍保持开启
/pomodoro status   查看当前群番茄钟状态
/pomodoro config   查看当前配置
/pomodoro help     查看帮助
```

### 中文命令

```text
/番茄钟 开启
/番茄钟 关闭
/番茄钟 开始
/番茄钟 停止
/番茄钟 状态
/番茄钟 配置
/番茄钟 帮助
```

## 默认配置

- 专注时间：25 分钟
- 短休息：5 分钟
- 长休息：15 分钟
- 每 4 个番茄进入一次长休息
- 默认不在 AstrBot 启动后自动恢复番茄钟循环，避免打扰群聊

可在 AstrBot 插件设置页修改 `_conf_schema.json` 中定义的配置。

## 使用建议

1. 群内发送 `/pomodoro on` 或 `/番茄钟 开启`。
2. 发送 `/pomodoro start` 或 `/番茄钟 开始` 开始番茄钟循环。
3. 需要暂停时发送 `/pomodoro stop` 或 `/番茄钟 停止`。
4. 不再使用时发送 `/pomodoro off` 或 `/番茄钟 关闭`。

## 注意

- 插件依赖 AstrBot 运行环境。
- 本插件会使用 AstrBot 的 `unified_msg_origin` 主动向群聊发送提醒消息。
- 如果刚安装后无法主动提醒，请先在目标群内执行一次插件命令，让插件记录群上下文。
- 群内命令回复会自动 @ 使用命令的成员；如果当前平台事件无法提供发送者 QQ 号，则会退化为普通文本回复。
