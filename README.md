# AstrBot 多人番茄钟插件

一个可通过 QQ 群机器人开启、关闭和运行的 AstrBot 多人番茄钟插件。同一个群里，不同成员可以同时启动各自独立的番茄钟，到时间只 @ 对应成员。

## 功能

- 群内开启/关闭番茄钟功能
- 群内每个成员可开始/停止自己的独立番茄钟循环
- 命令回复会自动 @ 触发命令的人
- 番茄钟后台提醒会 @ 对应 timer 的触发者
- 同一个群里支持多个成员同时运行不同 timer
- 多个群独立运行
- 插件卸载时自动清理后台任务
- 状态持久化到 `data/state.json`
- 支持 WebUI 配置番茄钟时长

## 命令

### 英文命令

```text
/pomodoro on       开启当前群番茄钟功能
/pomodoro off      关闭当前群番茄钟功能，并停止群内所有 timer
/pomodoro start    为你开始一个独立番茄钟循环
/pomodoro stop     停止你的番茄钟循环，群内其他人不受影响
/pomodoro status   查看当前群状态和你的 timer 状态
/pomodoro config   查看当前配置
/pomodoro help     查看帮助
```

### 中文命令

```text
/番茄钟 开启
/番茄钟 关闭
/番茄钟 开始
/番茄钟 停止
/番茄钟 状态
/番茄钟 配置
/番茄钟 帮助
```

> 为避免 AstrBot 将 `/番茄钟 开始`、`/番茄钟 停止` 等子命令误匹配到父命令，本插件不再注册单独的 `/番茄钟` 或 `/pomodoro` 默认帮助入口。需要查看帮助请明确发送 `/番茄钟 帮助` 或 `/pomodoro help`。

## 默认配置

- 专注时间：25 分钟
- 短休息：5 分钟
- 长休息：15 分钟
- 每 4 个番茄进入一次长休息
- 默认不在 AstrBot 启动后自动恢复番茄钟循环，避免打扰群聊

可在 AstrBot 插件设置页修改 `_conf_schema.json` 中定义的配置。

## 使用建议

1. 群内发送 `/pomodoro on` 或 `/番茄钟 开启`。
2. 任意成员发送 `/pomodoro start` 或 `/番茄钟 开始`，即可开始自己的独立番茄钟。
3. 其他成员也可以同时发送 `/pomodoro start` 或 `/番茄钟 开始`，互不影响。
4. 需要暂停自己的 timer 时发送 `/pomodoro stop` 或 `/番茄钟 停止`。
5. 不再使用时发送 `/pomodoro off` 或 `/番茄钟 关闭`，会关闭当前群功能并停止群内所有人的 timer。

## 注意

- 插件依赖 AstrBot 运行环境。
- 本插件会使用 AstrBot 的 `unified_msg_origin` 主动向群聊发送提醒消息。
- 如果刚安装后无法主动提醒，请先在目标群内执行一次插件命令，让插件记录群上下文。
- 群内命令回复会自动 @ 使用命令的成员；番茄钟循环中的后台提醒会 @ 对应 timer 的成员。
- `/pomodoro stop` 和 `/番茄钟 停止` 只停止命令发送者自己的 timer，不会影响群内其他人的 timer。
- `/pomodoro off` 和 `/番茄钟 关闭` 会停止当前群内所有人的 timer。
- 如果当前平台事件无法提供发送者 QQ 号，则会退化为普通文本回复。
