# 更新日志

本文档记录 `astrbot_plugin_pomodoro` 的主要更新内容。

## v1.1.0 - 2026-06-09

### 新增

- 支持同一个群内多个成员同时开启独立番茄钟 timer。
- 每个成员的番茄钟独立计时、独立循环，到时间只会 @ 对应成员。
- `/pomodoro status` 与 `/番茄钟 状态` 会显示：
  - 当前群番茄钟功能状态
  - 当前群内正在运行的 timer 数量
  - 命令发送者自己的 timer 状态
  - 命令发送者已完成的番茄数
- 支持从旧版单群单 timer 状态迁移到新版多人 timer 状态。

### 变更

- `/pomodoro start` 与 `/番茄钟 开始` 现在只会为命令发送者创建独立 timer。
- `/pomodoro stop` 与 `/番茄钟 停止` 现在只停止命令发送者自己的 timer，不影响群内其他成员。
- `/pomodoro off` 与 `/番茄钟 关闭` 会关闭当前群番茄钟功能，并停止群内所有成员的 timer。
- 插件元数据更新为 `v1.1.0`，作者更新为 `Idris173`。

### 修复

- 修复后台番茄钟提醒不会 @ 触发者的问题。
- 修复 `/番茄钟 开始`、`/番茄钟 停止` 等子命令可能额外触发帮助消息的问题。
- 移除容易误触发的默认帮助入口 `/番茄钟` 与 `/pomodoro`。
  - 如需查看帮助，请明确发送 `/番茄钟 帮助` 或 `/pomodoro help`。

## v1.0.0 - 2026-06-09

### 新增

- 初始版本发布。
- 支持通过 QQ 群机器人开启/关闭番茄钟功能。
- 支持开始/停止番茄钟循环。
- 支持经典番茄钟配置：
  - 专注时长
  - 短休息时长
  - 长休息时长
  - 长休息间隔
  - 结束前提醒
- 支持 AstrBot WebUI 配置项。
- 支持状态持久化到 `data/state.json`。
- 支持中英文命令：
  - `/pomodoro on`
  - `/pomodoro off`
  - `/pomodoro start`
  - `/pomodoro stop`
  - `/pomodoro status`
  - `/pomodoro config`
  - `/pomodoro help`
  - `/番茄钟 开启`
  - `/番茄钟 关闭`
  - `/番茄钟 开始`
  - `/番茄钟 停止`
  - `/番茄钟 状态`
  - `/番茄钟 配置`
  - `/番茄钟 帮助`
